# MoodShot – My Pic, My Status

MoodShot is a multilingual photo quote mobile application built with Flutter.

## Features
- Pick image from gallery
- Add ready-made quotes
- Write custom text in multiple languages
- Urdu, Arabic, Hindi, English support
- Beautiful fonts and text styling
- Save image and share on social media

## Tech Stack
- Flutter (Dart)
- image_picker
- screenshot
- share_plus
